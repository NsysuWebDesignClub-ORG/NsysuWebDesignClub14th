Web程式設計

資管116級 楊雅安

課程簡介

本課程將帶領大家逐步掌握 HTML、CSS 與 JavaScript 的基礎能力，
並學習如何運用 GitHub 進行版本控制與協作開發。
每堂課都有安排實作練習，讓大家能一步步學會如何上架專屬的網站。即使完全沒有程式基礎，也能輕鬆上手喔！
